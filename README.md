# Lokalt oppsett

For å kjøre opp løsningen lokalt:

## Fra IntelliJ
Kjør [main-metoden](src/test/java/no/nav/[pakkenavn]/api/TestLauncher.java)

## Bygging og publisering

For å bygge imaget, kjør `sh build.sh`. Se `sh build.sh --help` for alternativer.
